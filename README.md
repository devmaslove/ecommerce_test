# ecommerce_test

Верстка магазина на Flutter.

Содержание:
- Экран «Main»: выбор категории, фильтрация по двум параметарам
- Экран  «Product Details»: карусель, инфо, добавление в корзину
- Экран «My Cart»: корзина, изменение количества (0 - удаление)
